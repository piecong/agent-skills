# Frontend Team Skills

这个目录包含 Frontend 团队（前端开发团队）专属的 skills。

## 团队职责

Frontend 团队负责前端技术体系建设，包括：
- 页面开发：Web 应用 UI 页面的开发与交互实现
- 组件库建设：开发和维护内部通用组件库
- 工程化与构建：前端构建工具、脚手架及 CI/CD 流程维护
- 性能优化：页面加载性能与渲染性能优化
